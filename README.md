# Notebooks para Bioacústica

Notebooks Jupyter interativos para análise bioacústica de gravações de campo.

Todos os notebooks executam em [Google Colab](https://colab.research.google.com/) e funcionam com arquivos de áudio armazenados no Google Drive ou [Arbimon](https://arbimon.org).

1. Abra um notebook da pasta `pt-BR/`
2. Clique em "Open in Colab" ou copie a URL do notebook para o Colab
3. Siga as instruções passo a passo em cada notebook
4. Nenhuma instalação necessária — tudo executa na nuvem

# Bioacoustic Notebooks

Interactive Jupyter notebooks for bioacoustic analysis of field recordings.

All notebooks run on [Google Colab](https://colab.research.google.com/) and work with audio files stored on Google Drive or [Arbimon](https://arbimon.org).

1. Open a notebook from the `en/` folder
2. Click "Open in Colab" or copy the notebook URL to Colab
3. Follow the step-by-step instructions in each notebook
4. No installation required — everything runs in the cloud
