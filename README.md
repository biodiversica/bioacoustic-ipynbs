All notebooks run on [Google Colab](https://colab.research.google.com/) and work with audio files stored on Google Drive or [Arbimon](https://arbimon.org).

1. Click on a notebook link below to view it on GitHub
2. Copy the notebook to your Colab or local Jupyter environment
3. Follow the step-by-step instructions in each notebook
